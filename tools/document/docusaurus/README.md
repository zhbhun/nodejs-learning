# 安装使用
- `npm install --global docusaurus-init`
- `docusaurus-init`
- `mv docs-examples-from-docusaurus docs`
- `cd website`
- `mv blog-examples-from-docusaurus blog`
- `npm start`

# 网站结构
```
root-of-repo
├── docs
└── website
│   └── blog
│   └── core
│       └── Footer.js
│   └── node_modules
│   └── package.json
│   └── pages
│   └── sidebars.json
│   └── siteConfig.js
│   └── static
```
